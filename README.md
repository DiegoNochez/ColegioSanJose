# 🎓 Sistema de Gestión Escolar - Colegio San José

Proyecto ASP.NET Core MVC + Entity Framework Core + MySQL

## 🚀 Descripción
Aplicación web para la administración académica del Colegio San José.  
Permite gestionar **alumnos**, **materias** y **expedientes académicos**, incluyendo calificaciones, reportes y dashboard de estadísticas.

## 🧩 Tecnologías utilizadas
- **ASP.NET Core 8.0 MVC**
- **Entity Framework Core (Pomelo MySQL)**
- **Bootstrap 5.3 + Bootstrap Icons**
- **C# / Razor Pages**
- **Chart.js** (para los gráficos del dashboard)

## 📋 Funcionalidades principales
- CRUD completo de alumnos, materias y expedientes
- Validaciones de datos con DataAnnotations
- Cálculo de promedios generales y por grado
- Dashboard con estadísticas visuales
- Diseño moderno y responsivo
